# About
This game was made for [Juniper Dev's Very Serious Game Jam](https://itch.io/jam/theveryseriousjuniperdevgamejam), though unofficially as I was in the middle of exams during the real jam.

The concept is simple: there are machines that let you change your gravity direction, and you have to use them to platform to the end of the level!

This is our first time participating in a game jam and our first time making something like this in general (and also my first time using Godot), apologies for the rough edges!

We also ran out of time so we only managed to make 1 level...


The game is made in [Godot](https://godotengine.org/), while the art was made using [Aseprite](https://www.aseprite.org/)!

# Install instructions
Just extract the zip for your preferred platform and run the executable

## Controls
WASD + Space and F
